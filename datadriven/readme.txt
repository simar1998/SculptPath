This project is aimed to create a non planar 3d printer toolpath generator.

The dataset for stl files used to train the model is here https://zenodo.org/records/1098527/files/all-stl.zip?download=1

Additional information about the dataset can be found here
https://www.mdpi.com/2306-5729/3/1/5

Python is used to create train the model and c++ is used to generate dataset for ray intersection dataset and the gcode for training purposes.
